== Description ==
LawGit - team presents absolutely fresh and powerful WordPress theme. It is fully responsive for all kind of devices like Desktop & Mobile.
== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Choose seoaal theme file and install.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Plugins -> LawGit Core to activate and customize to your taste.
